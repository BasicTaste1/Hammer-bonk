# Hammer Bonk
Plays a squeaky toy sound and displays above head text when you or a teammate hit with a DWH or Elder Maul instead of using their special attacks.

Allows for enabling of squeaking sound and/or overhead chat message.\
Volume adjustor for sound effect.\
Customizable overhead text.\
Duration adjustor for overhead text.
